## 기준

## 점검 항목
### 개인정보
- 유아 이름·진단명·약물명·보호자 연락처: 입력 필드/DB 컬럼/변수/로그에 **존재 금지**.
- 취약 유아 정보는 숫자 집계값만. 자유텍스트에 PII 유도 없음.
- **AI/외부 API 전송 페이로드에 PII 0건**(화이트리스트 확인).

### 안전 표현
- 의료 진단·질병 단정·약물 권고 없음.
- 공식기관 지시보다 우선한다는 표현 없음 → 공식기관·119 우선 안내.
- "AI가 위험 예측" 단정 대신 "대응 우선순위 제안".
- 결과/안내 화면 고정 안전 문구 노출.
- 학부모 안내문 공포 표현 없음 + 조치 중 안정감 + 구체 행동.
- 허위 연락처/기관명 생성 없음.

## 산출물(반환 형식)
- 위반/우려 목록: 각 항목 **파일·위치 + 근거 원칙 + 수정 제안**.
- 위반 없으면 "통과" + 점검 범위 명시.

**금지**: 파일 수정·명령 실행. 검토 결과만 반환한다.
