## Fallback 시연(차별화 포인트, 선택 30s)
- 네트워크/AI 차단 상황 가정: `USE_SAMPLE_FALLBACK=true` 또는 키 제거.
- 동일 흐름이 **샘플 결과 배지**와 함께 멈춤 없이 동작함을 시연 → "장애에도 시연이 멈추지 않는다".

## 안전·개인정보 어필 포인트(심사 강조)
- 입력 화면에 유아 이름/진단명/연락처 **필드 자체가 없음**.
- AI 전송 데이터에 PII 0건(화이트리스트).
- 의료 진단 없음, 공식기관 우선 표기.

## 백업 플랜(데모 사고 대응)
- AI 지연 → 자동 fallback 결과로 진행(설명 그대로 사용).
- DB 오류 → in-memory 샘플 시드로 진행.
- 완전 오프라인 → `USE_SAMPLE_FALLBACK` 모드 사전 기동.

## P6 공공 API 시연 포인트(선택)
- `/plan/new/message` → **"실시간 조회"** 탭 → "가져오기" 버튼 → 행안부 API 또는 샘플 재난문자 목록
- 기관 등록 주소 입력 → **"자동 채움"** → Kakao Geocode API 좌표 자동 입력
- AI 결과 `reflected_evidence`에 실측 기온·체감온도 반영(KMA 초단기예보)
